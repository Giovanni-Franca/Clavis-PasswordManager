# PT-BR 
O aplicativo Clavis tem como finalidade oferecer uma forma de gerar novas senhas aleatórias e amarzen-las para seus úsuarios, sendo pensado naqueles que tem dificuldades de criar novas senhas ou dificuldades para lembrar senhas existentes, garantindo segurança e confiabilidade.

# EN
The Clavis application aims to provide a way to generate and store new random passwords for its users, designed for those who have difficulty creating new passwords or remembering existing ones, ensuring security and reliability.
